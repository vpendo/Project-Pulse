# Video Assets

## Hero Background Video

To add a video background to the homepage:

1. Place your video file(s) in this directory (`public/videos/`)
2. Name it `hero-background.mp4` (or `hero-background.webm`)
3. Recommended specifications:
   - **Format**: MP4 (H.264) or WebM
   - **Resolution**: 1920x1080 (Full HD) or higher
   - **Duration**: 10-30 seconds (will loop)
   - **File Size**: Keep under 5MB for better performance
   - **Aspect Ratio**: 16:9

## Video Sources

You can use free stock videos from:
- **Pexels Videos**: https://www.pexels.com/videos/
- **Pixabay Videos**: https://pixabay.com/videos/
- **Coverr**: https://coverr.co/

## Optimization Tips

1. Compress your video using tools like:
   - HandBrake (free)
   - FFmpeg
   - Online tools like CloudConvert

2. For better performance, provide both MP4 and WebM formats:
   - `hero-background.mp4` (for broader compatibility)
   - `hero-background.webm` (for better compression)

3. If you don't have a video, the page will fall back to a gradient background.
